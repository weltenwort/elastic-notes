Diagnosis of pipeline processor error

  • error.pipeline contains the pipeline containing the failing processor, such as "logs@custom"
  • error.pipeline_trace contains the call stack of involved pipelines, such as ["logs@default-pipeline", "logs@custom"]
  • error.processor_type contains the type of the processor, such as json
  • error.message contains details, but the format is diverse
  • the combination of error.processor_type and error.type is not sufficient to disambiguate the error.message format
